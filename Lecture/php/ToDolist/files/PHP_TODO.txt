PHP でTODOリストアプリを作ろう！

テーブル定義
tasks テーブル
 - id: int primary auto_increment (タスクの主キー)
 - title: varchar(255) not null タスクのタイトル
 - detail: text タスクの詳細
 - done: bool not null default false タスクの状態（完了 or 未完了）
 - user_id: int not null タスクを登録したユーザーのid

users テーブル
 - id: int primary auto_increment (ユーザーの主キー)
 - name: varchar(255) not null ユーザーの名前
 - email: varchar(255) not null ユーザーのメールアドレス(ログイン情報として使用)
 - password: varchar(255) not null ユーザーのパスワード（ログイン情報として使用）

実装する機能
【ユーザーログイン機能】
・メールアドレスとパスワードでログインする
・メールアドレスとパスワードの検証に成功したら、セッションにログイン中のユーザーの情報を記録する

【タスク追加機能】
・タスクのタイトルと詳細内容を入力して、追加できる
・追加されたタスクは、ユーザー固有のデータとしてデータベースに保存する（user_id をセットする）

【タスク一覧表示機能】
・ログイン中のユーザーが登録したタスクを一覧表示する
・別のユーザーが登録したタスクは表示されない

【タスク削除機能】
・登録されたタスクを削除できる
　・（+α）別のユーザーが登録したタスクは削除できない

【タスク編集機能】
・登録されたタスクのタイトルと詳細情報を編集できる
　・（+α）別のユーザーが登録したタスクは編集できない

【タスク完了機能】
・登録されたタスクの状態を変更できる
・完了のタスクは、一覧の中で「完了」と表示される
　・（+α）別のユーザーが登録したタスクは変更できない

【ユーザー登録機能】
・ユーザー名とメールアドレスとパスワードを入力してユーザー登録する
・登録済みのメールアドレスの場合は、ユーザー登録できない
